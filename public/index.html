Privacy

This page generates passwords entirely in your browser using window.crypto.getRandomValues() — a cryptographically secure source backed by your operating system's CSPRNG. Nothing is sent over the network. No analytics, no telemetry, no cookies. The page itself is served as static HTML/CSS/JS from a private nginx container.

The source code is public — verify the claim: github.com/AaronShemtov/pwd-frontend
